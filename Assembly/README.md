# Assembly

Materiale didattico e esercizi in Assembly ARM, organizzati per capitolo.

## Capitoli

1. `Capitolo 1` - introduzione a ARM, registri, istruzioni base e direttive.
2. `Capitolo 2` - shift/rotate, logica bitwise, moltiplicazioni e overflow.
3. `Capitolo 3` - flag CPSR, branching, if/while/for e controllo di flusso.
4. `Capitolo 4` - vettori, stringhe e stack.

## Altre directory

1. `EsempiEserciziProva` - tracce ed esercizi di prova con file `.s`.
